# `brilliant-msg`

Application-level library for passing rich objects between a host program and [Brilliant Labs](https://brilliant.xyz/) Halo and Frame devices, such as images, streamed audio, IMU data and rasterized text.

[Brilliant SDK documentation](https://docs.brilliant.xyz/frame/frame-sdk/).

## Features

* Provides high-level messages for sending text and images for display on Frame/Halo, receive photos, audio, IMU data and taps.
